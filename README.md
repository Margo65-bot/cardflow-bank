## 📖 О проекте

**CardFlow** — это backend-приложение для управления банковскими картами, разработанное на Java с использованием Spring Boot. Приложение предоставляет функционал для создания, просмотра и управления банковскими картами, а также выполнения переводов между собственными картами пользователя.

Проект реализован в рамках тестового задания.

---

## 🚀 Функциональные возможности

### 👑 Администратор
- Создание, блокировка, активация и удаление карт
- Управление пользователями (создание, изменение роли, удаление)
- Просмотр всех карт и всех пользователей
- Просмотр всех транзакций

### 👤 Пользователь
- Просмотр своих карт с фильтрацией и пагинацией
- Просмотр баланса карты
- Запрос на блокировку карты (статус `PENDING_BLOCK`)
- Переводы между своими картами
- Просмотр истории переводов по карте

### 🔐 Аутентификация и авторизация
- Вход с использованием JWT токенов
- Ролевая модель: `ADMIN` и `USER`
- Защита эндпоинтов с помощью `@PreAuthorize`
- Шифрование номеров карт (AES-256)

| Git | 2.0+ | `git --version` |

---

**1. Клонировать репозиторий**

```bash
git clone https://github.com/Margo65-bot/cardflow-bank.git
cd cardflow-bank
```

**2. Запустить приложение**

```bash
docker-compose up --build
```

**3. Проверить работу**

```bash
docker ps
curl http://localhost:8080/actuator/health
```

**4. Открыть Swagger UI**

```bash
http://localhost:8080/swagger-ui/index.html
```
